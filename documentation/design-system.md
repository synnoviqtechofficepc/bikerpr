# VeloWorks — Design System Reference

## Colors
| Token | Value | Usage |
|-------|-------|-------|
| `--clr-primary` | `#16A34A` | Brand, CTAs, links |
| `--clr-primary-dark` | `#15803D` | Hover states |
| `--clr-secondary` | `#F97316` | Accents, badges |
| `--clr-text` | `#0F172A` | Body text |
| `--clr-text-muted` | `#64748B` | Secondary text |
| `--clr-border` | `#E2E8F0` | Dividers |
| `--clr-bg-alt` | `#F8FAFC` | Section backgrounds |

## Button Classes
- `.btn--primary` — Green filled CTA
- `.btn--outline` — Green outline
- `.btn--ghost` — Bordered neutral
- `.btn--secondary` — Orange filled
- `.btn--sm` / `.btn--lg` / `.btn--xl` — Size modifiers
- `.btn--block` — Full width

## Badge Classes
- `.badge--success` — Green
- `.badge--warning` — Yellow
- `.badge--error` — Red
- `.badge--info` — Blue
- `.badge--neutral` — Grey
- `.badge--primary` — Green light
- `.badge--orange` — Orange light

## Status Components
- `.status-dot--active` — Pulsing green dot
- `.status-dot--pending` — Yellow dot
- `.timeline` — Horizontal step tracker
- `.vtimeline` — Vertical notes tracker
- `.repair-progress` — Animated progress bar

## Layout Helpers
- `.container` — Max 1280px, auto-centered
- `.container--narrow` — Max 800px
- `.section` — 96px vertical padding
- `.grid--2/3/4` — Responsive grid
- `.dashboard-layout` — Sidebar + main

## Component Patterns
- `.card` — Image card with hover lift
- `.service-card` — Icon + text service block
- `.tier-card` — Membership tier pricing
- `.testimonial-card` — Quote + author
- `.team-card` — Photo + bio
- `.booking-item` — Date + title list row
- `.stat-card` — Dashboard metric
- `.accordion-item` — Collapsible FAQ
- `.time-slot` — Booking grid cell
- `.service-select-card` — Radio-style selection
